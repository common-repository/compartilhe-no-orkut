Exibe um botão para compartilhar o conteúdo no Orkut no final de cada artigo.

== Description ==

O plugin Compartilhe no Orkut é uma ferramenta para exibir no final de cada arigo do Wordpress um botão para seus leitores compartilharem seu conteúdo no Orkut.

== Installation ==

1. Baixe e descompacte o plugin
2. Envie a pasta "compartilhe-no-orkut" para a pasta wp-content/plugins do seu Wordpress
3. Ative no painel de controle
